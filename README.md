# Media Accessibility

A goal of this task is to learn about accessibility and how can we make audio and video files accessible to everyone. So, it can be used by people with disabilities as well.

## Build With

-   HTML5
-   CSS3
-   VanillaJs
-   SASS
-   Bootstrap
